# 餐廳清單
一個簡單的餐廳清單網頁。

## 產品功能
- 使用者可以在首頁看到餐廳的簡單資料。
- 點擊餐廳資料可以查看該餐廳的詳細資訊。
- 使用者可以依照餐廳名稱分類進行搜尋。

## 安裝流程

專案資料clone到本地

```
$ git clone https://github.com/ddssads/restaurant-list.git

```
安裝npm

```
$ npm install
```

執行

```
$ npm run dev
```

終端機顯示表示成功執行
```
Server is running on http://localhost:3000
```
